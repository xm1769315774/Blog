# 创建新的 React 应用

电脑环境需要安装Node 版本大于 8.10 和 npm 版本大于 5.6

```cmd
npx creat-react-app 项目名称
	或
npm creat react-app 项目名称
```

# React和Vue的区别

react和vue一样都是UI框架，专注于负责前端UI界面渲染。

react是单向数据流，vue是双向数据流

+ react基于MVC模式

+ react本身会监听数据变化从而渲染视图，但不会监听视图改变来修改数据
