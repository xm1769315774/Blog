date: 2021-06-23

Node.js® 是一个基于 Chrome V8 引擎 的 JavaScript 运行时环境。

Node.js没有DOM、BOM的操作，基于ECMAScript标准基础语法及一些额外的API进行服务器的操作
例如：文件的读写，网络服务的构建，网络的通信等

特性：
一个Javascript运行环境
依赖于Chrome V8引擎进行代码解释
事件驱动
非阻塞I/O  (阻塞线程/浪费 CPU 循环等待，异步可以解决这种问题) 
轻量、可伸缩，适于实时数据交互应用
单进程，单线程
